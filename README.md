## 💼 核心项目介绍

### 项目一：智能题库转换与去重系统

**关键词**: `AutoHotkey` `Pandoc` `正则表达式` `模糊匹配` `Excel COM`

**项目描述**:
针对教育行业中将大量 Word 格式题库转换为标准化 Excel 格式的痛点，开发了一套全自动处理系统。该系统不仅能完成格式转换，还能智能识别题型、分离选项与答案，并通过高精度模糊匹配算法对题库进行去重和相似题检测。

**技术亮点与职责**:

1.  **复杂文本解析引擎**:
    - 使用 Pandoc 将 `.docx` 转换为纯文本，作为处理基础。
    - 编写了超过 500 行的文本清理和正则表达式规则，处理多种不规范的题型表述（如“- A ．”等）。
    - 实现了特殊符号（如 Mpa, ℃）的保护机制，确保技术类题目的准确性。

2.  **题型智能识别与处理**:
    - 通过关键词和上下文分析，自动分类单选题、多选题和判断题。
    - 对判断题的答案进行了智能反转逻辑处理（A/B 反转）。
    - 规范了多选题答案的格式（如从“正确答案：AB”转换为“A,B”）。

3.  **高精度模糊去重算法**:
    - 实现了两种可配置的模糊匹配算法：
        - **方案一（高精度）**: 采用**单字符滑动窗口**，匹配阈值设为 **70%**，确保只有高度相似的题目才被去重，适合严格去重场景。
        - **方案二（高性能）**: 采用**三字符滑动窗口**，匹配阈值设为 **50%**，适用于快速查找相似题，适合宽松匹配场景。
    - 支持生成带行号索引的比对报告，便于数据溯源和人工复核。

4.  **Excel 报表自动化生成**:
    - 通过 Excel COM 接口，将最终结果直接生成格式美观的 `.xlsx` 文件。
    - 自动设置表头（浅蓝色背景、加粗）、调整列宽，并支持分批写入以避免内存溢出。

**项目成效**:
将原本需要数天手动整理和校对的工作，缩短至**几分钟内自动完成**，准确率超过 **95%**，极大提升了题库管理和维护的效率。

### 项目二：大乐透数据全链路自动化分析平台

**关键词**: `UI自动化` `OCR集成` `像素检测` `数据处理` `算法`

**项目描述**:
一个集**数据采集、识别、计算、模拟**于一体的综合性数据分析平台。旨在通过自动化手段研究大乐透彩票，涵盖了从获取原始数据到生成分析报告的全过程，展现了全栈式自动化开发能力。

**技术亮点与职责**:

1.  **Robust 的 UI 自动化采集端**:
    - 针对微信小程序控件难以识别的问题，创新性地采用**像素颜色检测** (`PixelSearch`) 来定位“机选”、“清空”等按钮，不依赖控件 ID，适应性极强。
    - 实现了**多窗口并行控制**，自动排列窗口布局，显著提升了数据采集效率。
    - 集成 **ShareX** 进行精确的区域截屏，自动识别号码区域的上下边界。

2.  **OCR 集成与数据清洗**:
    - 调用 **Umi-OCR** 进行批量图片文字识别。
    - 编写了专门的文本清理函数，有效过滤 OCR 产生的乱码和垃圾行，确保后续处理数据的纯净度。

3.  **核心算奖引擎与矩阵分析**:
    - 使用 `Map` 数据结构构建了完整的奖金映射表，通过字符串匹配算法快速计算每注号码的中奖情况。
    - 实现了**矩阵拆分与重组算法**，能从基础组合衍生出不同层级的号码组合，用于深度分析。
    - 生成详细的统计报告，包括总注数、总奖金、平均奖金及号码出现频率排序。

4.  **组合数学与数据模拟**:
    - 编写了高效的**递归算法**，能够生成数千万量级的所有可能号码组合（如前区5码+后区2码的所有组合），为概率分析和策略研究提供了坚实的数据基础。
    - 采用**分批生成与写入**策略，有效解决了海量数据的内存占用问题。

**项目价值**:
这不仅是一个实用的数据分析工具，更是一个展示**全链路自动化能力**的典范，完美演绎了从 **“前端”交互 (UI自动化) → 数据获取 (OCR) → “后端”逻辑处理 (算奖/算法) → 结果呈现 (文件/报表)** 的完整闭环。

### 项目三：CSV 文件智能比较与报告生成工具

**关键词**: `数据处理` `差异比对` `算法优化` `Excel COM`

**项目描述**:
一个用于自动比较不同时期 CSV 数据文件（如搜索排名、商品价格列表）的专业工具，能够快速识别变化、计算差异并生成精美的 Excel 分析报告。

**技术亮点**:
- **线性时间复杂度算法**: 通过为第二个文件构建内存索引，实现了 **O(m+n)** 的比对效率，即使处理数十万行数据也能快速完成。
- **智能垃圾清理**: 内置强大的 `CleanText` 和 `IsGarbageLine` 函数，能有效过滤 CSV 中的损坏数据和 OCR 识别错误。
- **完整的 CSV 解析器**: 能正确处理带引号的字段、嵌套引号和字段中的逗号。
- **一键生成报告**: 自动将差异结果通过 Excel COM 生成格式化的 `.xlsx` 报告，并清晰分类（如“排名上升关键词”、“新增关键词”）。
